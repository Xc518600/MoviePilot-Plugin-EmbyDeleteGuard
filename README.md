# Emby删除兜底清理

MoviePilot V2 插件：监听 Emby/Jellyfin/Plex 删除类 Webhook，延迟复查媒体库残留文件、刮削文件、空目录和下载器任务。

默认启用“安全报告模式”：只报告残留，不删除文件或种子。需要自动清理时，请在插件设置中关闭安全报告模式并开启对应清理项。

## 功能

- 监听媒体服务器删除类 Webhook
- 延迟复查，给原清理插件先执行
- 检查残留媒体文件、字幕、NFO、图片等刮削文件
- 检查空目录
- 检查 qBittorrent / Transmission 残留种子
- 可选发送 `DownloadFileDeleted` 联动事件
- 可选自动清理刮削文件、空目录、媒体文件和种子

## 安全建议

首次使用建议保持默认配置：

- 安全报告模式：开启
- 清理刮削/字幕：开启
- 清理空目录：开启
- 清理残留媒体：关闭
- 删除残留种子：关闭

确认报告准确后，再按需开启自动删除。
